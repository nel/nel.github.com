Un nouveau défi en Ruby !

Ca y est, le DIMELO Contest est de retour.

Pour cette 4e édition, le principe est simple : faites avancer la communauté open-source ruby et Dimelo récompensera les deux meilleurs contributeurs.

Le contest est ouvert du 1er juillet 2014 00:00 au 1er octobre 2014 23h59 CEST

Plutôt Xbox One ou PS4 ?

A vous de choisir !
#TeamXboxOne ou #TeamPS4

Des règles simples

Tout rubyiste peut participer. Pour gagner, il faut :

  • écrire un patch en ruby pour un projet ruby open-source, hébergé sur github et comptabilisant au moins 20 Stars,
  • soumettre une pull request auprès des mainteneurs du projet,
  • obtenir la validation de sa pull request avant la fin du contest,
  • nous envoyer le lien vers la pull request à contest at dimelo.com, avant la fin du contest.

2 gagnants seront tirés au sort lors du premier événement Paris.rb suivant la fin du contest.

Des questions ? Voir la section contact.

Les détails du défi

Projets acceptés

Les projets open-source hébergés sur github

  • Le projet doit être un projet (majoritairement) Ruby
  • Le projet doit être hébergé sur Github et avoir au moins 20 "stars"
  • Le projet doit être Open-Source

Le tirage au sort

Comment allons-nous déterminer les gagnants ?

  • Les 2 gagnants seront tirés au sort parmi les personnes qui auront vu leur pull requests acceptées pendant la durée du concours. Nous les contacterons pour qu'ils choississent #Xbox One Vs #ps4.
  • Dans un souci d'équité, chaque participant au défi n'a qu'une seule chance de gagner l'une des deux consoles, et ce même si plusieurs de ces pull requests ont été acceptées pendant la durée du Dimelo Contest. Pour mettre toutes les chances de votre côté, nous vous encourageons tout de même à soumettre plusieurs pull requests.
  • La pull request doit être du code Ruby (la documentation ne sera pas prise en compte).
  • Les pull requests rejetées ne seront pas prises en compte.
  • Seules les pull requests effectuées et acceptées entres les dates du contest seront prises en compte.
  • Le compte github qui fait la pull request doit exister AVANT le début de ce contest
  • Les équipes Dimelo et leurs proches ne peuvent pas participer au concours.
